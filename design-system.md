# Design System

## Cores (HSL)
O sistema de cores utiliza variáveis CSS HSL para suportar temas (Light/Dark).

- **Primary**: `hsl(var(--primary))`
- **Secondary**: `hsl(var(--secondary))`
- **Destructive**: `hsl(var(--destructive))`
- **Muted**: `hsl(var(--muted))`
- **Accent**: `hsl(var(--accent))`
- **Popover**: `hsl(var(--popover))`
- **Card**: `hsl(var(--card))`
- **Background**: `hsl(var(--background))`
- **Foreground**: `hsl(var(--foreground))`
- **Border**: `hsl(var(--border))`
- **Input**: `hsl(var(--input))`
- **Ring**: `hsl(var(--ring))`

## Tipografia
- **Sans (Corpo)**: Inter, sans-serif
- **Heading (Títulos)**: Outfit, sans-serif

## Bordas (Radius)
Utiliza variáveis para consistência.
- **Default**: `var(--radius)`
- **MD**: `calc(var(--radius) - 2px)`
- **SM**: `calc(var(--radius) - 4px)`

## Animações
- **Shimmer**: `shimmer 2s infinite`
